## Install
		git clone https://github.com/GitIndonesia/GitIndonesia.github.io
		sudo gem install jekyll jekyll-paginate octopress
		octopress new post "your new post"
		jekyll build
		jekyll serve

## Deploy
		git add -u
		git add .
		git commit -m 'Message'
		git push
