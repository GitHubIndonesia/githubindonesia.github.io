Git?

Yang belum tau, Git adalah perangkat lunak pengontrol versi (VCS) untuk pengembangan program individual dan bersama-sama.

Cepat

Buat program jadi cepat kalo pake git, bisa kolaborasi sama developer-developer lain. Bisa liat kode sebelumnya, Revisi kode, dll. Keren kan.

Open Source

Banyak orang di belahan dunia yang membuat program terbuka dan berkolaborasi menggunakan git. Contoh yang lumayan banyak di GitHub .
